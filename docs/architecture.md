# Архитектура

## Диаграмма инфраструктуры

Пользователь → Интернет → Security Group → ВМ (Ubuntu 22.04)
↓
VPC Network
↓
Подсеть 192.168.10.0/24

## Компоненты

### Сеть
- **VPC Network**: Изолированная среда
- **Subnet**: Подсеть в зоне ru-central1-a (по умолчанию)
- **Security Group**: Правила доступа по SSH (22) и HTTP (80)

### Вычисления (значения по умолчанию)
- **Виртуальная машина**: 2 vCPU, 2 GB RAM
- **Образ ОС**: Ubuntu 22.04 LTS
- **Диск**: 20 GB SSD

### Безопасность
- SSH ключи через метаданные
- NAT для публичного IP
- Ограничение доступа через Security Group
